NOM
     analog -- analyse de log Apache

SYNOPSIS
     analog [-e] [-t heure] [-g output.dot] input.log

DESCRIPTION
     Affiche des statistiques et produit des graphiques utiles pour l'analyse
     du trafic sur un serveur a partir de logs Apache. Differentes options de
     filtrage sont disponibles.  Les statistiques des 10 ressources les plus
     demandees sont affichees.

OPTIONS
     Les options suivantes sont disponibles :

     -e      Exclut les documents de type image (jpg, png, ...), CSS, et
             JavaScript.

     -t      Exclus tous les hits qui ne sont pas dans le creneau horaire
             specifie [ heure ; heure + 1[.  L'argument heure doit etre un
             entier compris entre 0 et 23 inclus.

     -g      Produit un fichier de graphe GraphViz enregistre au chemin speci-
             fie dans output.dot de toutes les ressources.

VALEURS DE RETOUR
     Le programme analog retourne les valeurs suivantes a la fin de son execu-
     tion :

     0       l'execution s'est deroulee normalement.

     1       le programme a rencontre une erreur (acces fichier, ...).
